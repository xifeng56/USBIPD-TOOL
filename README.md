This is a WSL USB sharing program\n
Use config.txt add config\n
Its regular expression is as follows\n
@$"^(?!.*Attached)(\d+-\d+)\s.*(?:{newLine})"\n
First, install usbipd\n
Then, use the usbipd list command in the terminal to find device.\n
